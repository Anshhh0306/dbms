# Database Normalization Presentation

A clean, modern, scrollable presentation on the topic of Database Normalization (DBMS). It includes an explanation of the "Why", an interactive timeline of Normal Forms (1NF up to 5NF), and a visual structural transformation example showing Unnormalized data versus Students, Courses, and Enrollments tables.

## Viewing the Presentation
Just open `index.html` in any modern web browser.

## Tech Stack
- Vanilla HTML5
- CSS3 (Flexbox, Grid, CSS Variables)
- Vanilla JavaScript (Intersection Observer for scroll animations)

## Hosting on GitHub Pages
This site is designed to be hosted out-of-the-box on GitHub Pages. Since the entry file is `index.html` in the root directory, enabling GitHub Pages on the `main` or `master` branch will instantly publish the presentation.
